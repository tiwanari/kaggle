# Noets
## Get started...
1. Sign up at https://www.kaggle.com.
1. Install [`kaggle`](https://github.com/Kaggle/kaggle-api) command.
    * Run `pip install kaggle`
1. Check `Competitions` and choose an Active competition.
    * Starting with projects that don't have any price rewards like $10,000 :P
